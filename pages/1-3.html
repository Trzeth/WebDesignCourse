<ol> <ul> <li> <table>标签 列表 / 表单

<ol> 标签

定义和用法

<ol> 标签定义有序列表。

实例

有序 HTML 列表：

  1. Coffee
  2. Tea
  3. Milk
    <ol>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
    </ol>

<ul> 标签

定义和用法

<ul> 标签定义无序列表。

实例

无序 HTML 列表：

  • Coffee
  • Tea
  • Milk
    <ul>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
    </ul>

<li>标签

定义和用法

<li> 标签定义列表项目。

<li> 标签可用在有序列表 (<ol>) 和无序列表 (<ul>) 中。

实例

  1. Coffee
  2. Tea
  3. Milk

  • Coffee
  • Tea
  • Milk
    <ol>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
    </ol>
     
    <ul>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
    </ul>

<table>标签

定义和用法

<table> 标签定义 HTML 表格。

简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成。

tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元。

更复杂的 HTML 表格也可能包括 caption、col、colgroup、thead、tfoot 以及 tbody 元素。

实例

一个简单的 HTML 表格，包含两行两列：

Month Savings
January $100 
    <table border="1">
       <tr>
         <th>Month</th>
         <th>Savings</th>
       </tr>
       <tr>
         <td>January</td>
         <td>$100</td>
       </tr>
    </table>
